## 🛠️ Development

```sh
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run tests
pnpm test

# Lint and format
pnpm check

# Type check
pnpm typecheck
```

> All tasks run through [Turborepo](https://turbo.build) — only what changed gets rebuilt.
